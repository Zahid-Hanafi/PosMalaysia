Section 6.0

Details of SAP Enterprise Resource Planning (ERP)

Implemented around 2013, the SAP ERP system serves as the centralized digital backbone for Pos Malaysia's HR and Finance departments.

👥

6.2.1 HR Management (HCM)

  • • Employee Lifecycle: Manages recruitment, promotions, and retirement processes.
  • • Payroll Processing: Automatically calculates monthly salaries for 15,000 employees.
  • • Statutory Deductions: Calculates EPF, SOCSO, and tax deductions accurately.
  • • Documentation: Generates annual EA forms for LHDN tax submissions.
💰

6.2.2 Financial Management (FI/CO)

  • • Billing Engine: Produces approximately 100,000 invoices per billing cycle.
  • • Ledger Control: Manages accounts receivable, accounts payable, and general ledgers.
  • • Budgeting: Controls cost centre allocations to monitor departmental spending.
  • • Compliance: Ensures operations meet strict financial regulatory requirements.
🔗

6.2.3 System Integration

  • • Interconnectivity: Currently operates with 165 active system interfaces.
  • • Data Exchange: Uses APIs for real-time transfers and FTP for batch processing.
  • • Key Links: Pulls data directly from the Billing and Customer Relationship Management (CRM) systems.
📊

6.2.4 Report Generation

  • • Management Insights: Converts raw transactions into structured management reports.
  • • Transparency: Enables managers to drill down from summaries to individual transactions.
  • • Audit Readiness: Provides crucial documentation trails required for internal and external audits.
Section 7.0

Information System Components

The SAP ERP system relies on six fundamental components to operate smoothly across the nation.

7.1.1 Server Infrastructure

The hardware is divided into Production Servers for live, day-to-day operations and Non-Production Servers which are strictly used for testing new updates safely.

7.1.2 Cloud & On-Premise

Pos Malaysia utilizes a hybrid setup. They maintain physical data centre servers but have also migrated heavily to the AWS Cloud since 2022 to improve system reliability.

SAP ERP System

The core business software, primarily utilizing the highly customized HCM (HR) and FI/CO (Finance) enterprise modules.

Interfaces & Access

End users access the system through the SAP GUI desktop application. It links to 165 external systems natively through advanced APIs.

Platform

Data is managed by an enterprise-grade RDBMS SQL database, highly structured to handle complex relationships and rapid queries.

Data Stored (11TB)

It acts as the single source of truth, securely storing 11 Terabytes of HR profiles, payroll records, financial ledgers, and deep operational logs.

LAN

Local Area Network used within headquarters to provide high-speed, direct connections to the main servers.

WAN

Wide Area Network utilized to securely connect over 3,500 branches and touchpoints nationwide.

Protocols

Dependent on APIs for instant, real-time data syncs, and FTP for transferring massive overnight batch files.

Support Team

Maintained by a core IT team of 8 Administrators, alongside specialized external SAP vendors for major troubleshooting.

End Users

Used by approximately 200 Active Users, primarily HR and Finance executives who require formal SAP certification to operate the complex GUI.

Operations

Strict schedules govern when massive payroll batches or monthly billing cycles can be executed.

Monitoring & Backup

Real-time dashboards track server CPU health, while the 11TB database undergoes strict daily backup routines.

High Availability

Disaster Recovery procedures dictate failover plans to switch to backup servers instantly if primary systems fail.

📥 Input

  • Manual Entry: Complex HR and financial data typed directly into the SAP GUI by authorized staff.
  • API Data Transfer: Data pulled automatically from 165 integrated systems (such as daily parcel numbers from the Logistics system).
  • Validation: The SAP system applies strict rules to check data formats before accepting them, preventing "garbage in, garbage out".

⚙️ Processing

  • Payroll Computations: Automatically calculates gross-to-net salaries, EPF, and SOCSO for 15,000 employees simultaneously.
  • Financial Transformation: Converts thousands of raw daily transactions into structured, compliant accounting ledgers.
  • Automated Billing: Processes up to 100,000 billing records using complex, pre-defined corporate pricing rules.

📤 Output

  • Invoices: Roughly 100,000 highly detailed customer invoices generated and dispatched per cycle.
  • HR Documentation: 15,000 monthly electronic payslips and annual EA forms required for LHDN submissions.
  • Strategic Reports: Generates management dashboards, departmental cost centre summaries, and financial ageing reports.

💾 Storage

  • Centralized RDBMS: A massive 11 Terabyte SQL database securely housing a decade of operational data.
  • Access Control: Highly restrictive, role-based permissions ensure that employees only access the specific data required for their jobs.

🔄 Feedback

  • Performance Monitoring: IT administrators continuously track CPU usage, memory loads, and database query speeds.
  • Error Logs: Analysing failed transactions or slow processes to fix bugs and accurately plan future server upgrades.
