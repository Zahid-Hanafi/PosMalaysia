About the System

SAP Enterprise Resource Planning

The central information system of Pos Malaysia Berhad, implemented around 2013 to centralise HR and Finance operations.

📊

SAP ERP Platform

Global #1 ERP system used by Pos Malaysia to integrate Human Resource and Financial operations across all branches natively.

🗄️

RDBMS SQL Database

Stores 11 terabytes of structured data including employee records, financial transactions, and operational logs securely.

🔗

165 Integrations

Connects seamlessly with 165 internal and external systems via APIs and FTP to ensure data flows automatically without manual entry.

🖥️

SAP GUI Frontend

200 active users access the system through SAP's desktop application using strict role-based access control.

☁️

Hybrid Infrastructure

Runs on 7 backend servers combining physical on-premise hardware and AWS Cloud for high availability and disaster recovery.

💰

RM1.6M Annual Cost

Managed by a core IT team of 8 staff and external vendors, with licensing costs of roughly RM1.6 million annually.
